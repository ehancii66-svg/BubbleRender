# 实时渲染肥皂泡上出现的虹彩颜色

**Namjung Kim and Kyoungju Park**
cgnamman@naver.com, kjpark@cau.ac.kr

实时渲染肥皂泡上出现的虹彩颜色的新技术。配置了基于垂直和平行于入射面偏振光的光学相位反射率。在这个框架上，使用光栅化来实现主光线的相交，并有效地近似折射和反射以实现实时性能。此外，使用基于GPU的Perlin噪声来模拟皂膜之间的晃动效果。这使得在实时动态环境中渲染具有视觉信服效果的肥皂泡成为可能。

## 1 引言

颜色是人类视觉感知的对应属性，颜色的物理规格也与物体、材质、光源等相关。物体的颜色是其表面、透射和发射特性的复杂结果，分为原始色和结构色。原始色是物体上的颜料颜色。结构色是由微结构中反射光线的光程差引起的。例如，肥皂泡的虹彩颜色尤其著名。由于肥皂泡具有由光干涉引起的美丽结构色，它们经常被用于儿童娱乐，也用于肥皂泡表演。因此，渲染 these 迷人的肥皂泡干涉色的方法一直是图形学界一个有趣的研究课题。

已经开发了几种渲染具有光干涉的肥皂泡的方法。由于这些方法采用传统的光线追踪算法来渲染肥皂泡，因此计算成本很高。引入了多阶段光线追踪算法，并使用多次反射和折射来实现光干涉。物理方法即使在大入射角下也能创建逼真图像，但它不能用于实时应用。

Kuck的技术提供了光线追踪和气泡动力学的适当近似。但是Kuck的模型仍然无法达到实时渲染的性能。引入了基于GPU的肥皂泡实时渲染算法。但是这种方法不能产生正确的干涉颜色。Durikovic引入了基于GPU的光谱渲染，它展示了在有限颜色下的实时性能。已经开发了基于GPU的具有令人信服结果的实时干涉渲染算法。但是这种方法忽略了肥皂泡的折射，导致了不自然的折射效果。此外，还开发了基于GPU的实时结构色渲染技术。虽然这项技术渲染了包括肥皂泡干涉色在内的各种结构色，但它高度依赖于用户指定的手工纹理。因此，这些技术不适合像游戏这样的实时动态环境。

本文提出了一种无需用户手动输入即可实时渲染肥皂泡上出现的虹彩颜色的新方法。我们结合了光学相位反射率和CIE-XYZ颜色转换，并产生了令人信服的结果。此外，我们的框架有效地近似了折射和反射。这使得在游戏等实时动态环境中渲染具有视觉上令人信服结果的肥皂泡虹彩颜色成为可能。

我们的方法流程图如图1所示。第2节介绍了如何使用GPU光栅化加速主光线相交。第3节介绍了根据光学相位反射率和CIE-XYZ颜色转换的干涉映射。第4节介绍了如何近似折射。第5节介绍了反射映射，第6节介绍了晃动效果。第7节给出了模拟结果。最后，第8节描述了结论和未来的工作。

## 2 光栅化

光线追踪执行大量计算，以在3D空间中找到光线与物体表面之间的交点。为了减少寻找主光线交点的计算开销，我们使用针对图形硬件优化的管线光栅化方法。我们如等式(1)调整投影设置，以匹配光线追踪主交点和光栅化的结果。光栅化的投影角是从相机和图像平面之间的关系有效地计算出来的。

投影角 = 2 * tan((图像平面宽度 / 2) / 相机的Z轴距离) (1)

## 3 干涉映射

光具有双重性质，即粒子性和波动性。波动模型能够很好解释的现象之一是干涉，这也是肥皂泡呈现虹彩颜色的来源。因此，我们将光干涉作为基于光谱的方法来处理。我们将入射光处理为采样光谱。每个采样光谱由以下两个阶段计算得出。

### 3.1 光学相位反射率计算

肥皂泡的虹彩颜色在阳光等高辐射环境中非常明显且色彩丰富。因此，我们采用了光学界针对高辐射单色光和薄膜的光学方程。根据这种方法，如果薄膜被光谱组成为 I_in(λ) 的光源照射，则反射光强度由等式(2)给出。

I_r(λ) = R_λ * I_in(λ) (2)

R_λ 是反射率，由等式(3)给出。

R_λ = R_s^2 * (1 - cos φ) / (1 + R_s^4 - 2 * R_s^2 * cos φ) + R_p^2 * (1 - cos φ) / (1 + R_p^4 - 2 * R_p^2 * cos φ) (3)

R_s 和 R_p 分别是垂直和平行于入射面的偏振光反射的振幅系数，由等式(4)计算。

R_s = (cos θ - √(n^2 - sin^2 θ)) / (cos θ + √(n^2 - sin^2 θ))
R_p = (n^2 * cos θ - √(n^2 - sin^2 θ)) / (n^2 * cos θ + √(n^2 - sin^2 θ)) (4)

θ 是入射光的入射角。反射波之间的相位差由等式(5)给出。

φ(λ, θ) = (4πd / λ) * √(n^2 - sin^2 θ) (5)

n 是皂膜的折射率， d 是皂膜的厚度， λ 是入射光的波长。

### 3.2 光谱颜色转换

在计算出反射率强度后，将每个波的强度转换为CIE-XYZ颜色，以生成有关光波属性的适当颜色。首先，我们计算傅里叶基函数和入射波光谱。然后，我们使用关于绘制的颜色匹配函数值的积分来计算CIE-XYZ颜色转换。图2显示了颜色匹配函数，等式(6)显示了积分公式。最后，我们将XYZ颜色转换为RGB颜色，并将范围限制在0到1之间。

X = ∫ I(λ)x(λ)dλ
Y = ∫ I(λ)y(λ)dλ
Z = ∫ I(λ)z(λ)dλ (6)

## 4 折射映射

当光线进入其他介质时，光线会根据斯涅尔定律改变光路。图3展示了这种关系。折射向量可以从斯涅尔定律推导出来，等式(7)显示了根据折射率的折射向量。

r = (-η(N·I) - √(η^2(N·I)^2 - η^2 + 1))N + ηI

如果从内部到外部的射出点是通过相交测试计算的，那么计算成本会非常高。由于单个肥皂泡有四次折射，计算成本是一个更为重要的问题。因此，我们通过部署 Wyman 的近似技术来有效地近似这个过程。

在人眼位置，我们寻找肥皂泡正面和背面深度值 (d_E) 的差异。并生成肥皂泡背面的法线贴图。然后在 P_1 + N_1 位置获取正面和背面的深度值差异 (d_N)。利用等式(7)从 E 和 N_1 获取 T_1。

P_2 = P_1 + d * T_1
d = (d_E + d_N) / 2 (8)

我们使用等式(8)获取从肥皂泡内部到外部的光线射出位置 (P_2)。将 P_2 乘以定义在眼睛位置的投影矩阵，投影到背面法线贴图上。然后我们在射出位置获取法向量 N_2 并获取折射向量 T_2。最后 T_2 用于环境贴图的折射。

## 5 反射与 HDRI 映射

通常菲涅尔项被广泛用于表示诸如肥皂泡等透明物体表面的反射效果。但是菲涅尔项具有相对较高的计算成本。因此，我们使用菲涅尔项的近似方法来提高渲染性能。等式(9)显示了此近似项，θ 是入射光与折射表面法向量之间的夹角。

R(θ) = R_0 + (1 - R_0)(1 - cos θ)^5 (9)

传统的成像系统在表示真实世界的辐射分布方面存在局限性。HDRI（高动态范围图像）是克服这一局限性的创新技术。为了表示真实世界的辐射，我们使用实时 HDRI 纹理映射技术构建 HDR 环境立方体贴图。因此，通过使用环境映射技术，自然光的辐射从肥皂泡反射和折射。

## 6 晃动模拟

肥皂泡由肥皂分子和水分子组成。在皂膜中，两个平行的肥皂分子层夹着一层水分子。因为水分子在外力（如风和重力）作用下相对于肥皂分子很容易移动，肥皂泡会因水分子运动而产生晃动效果。我们利用基于 GPU 的 Perlin 噪声技术，通过改变移动肥皂泡的厚度来近似晃动效果。晃动效果是在片段程序中关于片段的每个厚度计算的。

## 7 模拟结果

我们使用配备了 Intel i7 CPU 2.80Ghz、Nvidia GTX260 图形硬件的计算机，分辨率为 1024x1024。开发环境是基于 Window7、OpenGL 和 GLSL 的 Visual C++。每个肥皂泡由包含 10,242 个顶点和 20,480 个三角形的球体模型组成。图6显示了两个肥皂泡的渲染结果，总共 40,960 个三角形以平均每秒 76.1 帧的速度渲染。图7显示了 56 个肥皂泡的渲染结果，总共 1,146,880 个三角形以平均每秒 24.5 帧的速度渲染。

## 8 结论

本文提出了一种考虑光干涉的肥皂泡实时渲染方法。我们利用与入射面偏振相关的光学相位反射率和 CIE-XYZ 颜色转换，来渲染肥皂泡上出现的虹彩颜色。为了提高渲染性能，我们包含了几种方法，即基于 GPU 的光栅化方法以及反射和折射的有效近似。并且我们使用基于 GPU 的 Perlin 噪声来模拟皂膜之间的晃动效果。所提出的系统在实时动态环境中提供了考虑光干涉的肥皂泡令人信服的渲染结果。对于未来的工作，我们希望模拟排液现象并实现更准确的干涉效果。
