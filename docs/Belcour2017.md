# 微面元理论的实用扩展：可变虹彩的建模

**Laurent Belcour and Pascal Barla**  
ACM Transactions on Graphics (SIGGRAPH 2017), Vol. 36, No. 4, Article 65

---

## 一、论文动机与定位

### 1.1 背景

现实世界中的很多材质呈现 **goniochromatic（角度变色）** 效应——从不同方向观察时颜色会改变。这种虹彩效应广泛存在于：

- 肥皂泡、油膜、酒精痕迹
- 鸟类羽毛、昆虫甲壳、蛇皮、水果表皮
- 汽车漆、皮革涂层、氧化金属表面

虹彩的物理来源有两类：**衍射**（微观结构与波长相当的周期性）和**薄膜干涉**（厚度接近可见光波长的薄膜层）。本文聚焦于后者。

### 1.2 现有方案的问题

在图形学中，微面元 BRDF 的颜色来源通常是 **Fresnel 反射项**，它与波长相关但不产生干涉。要为微面元模型加入薄膜干涉，直接用光谱渲染需要每像素数十甚至上百个波长采样（参见原论文 Figure 2）——这在交互式编辑、空间变化参数等实际场景中完全不可行。

**Smits & Meyer (1992)** 是最早在图形学中处理薄膜干涉的工作，但它的模型有四个关键局限：
- 仅适用于完美光滑表面
- 不考虑偏振
- 不考虑薄膜内多次反射
- 不支持导体基底

## 二、论文核心贡献

本文的贡献一句话概括：

> **用 Airy 求和正确建模薄膜干涉，然后通过解析预积分在频域中一次性完成光谱积分，使得在 RGB 渲染引擎中能够以极小的开销获得接近光谱级精度的虹彩效果。**

具体而言：

1. **Airy 反射项** — 替代经典 Fresnel 项，完整建模薄膜内所有阶次的反射和干涉（包括偏振和导体相移）
2. **解析光谱预积分** — 利用 Airy 反射项在傅里叶空间是 Dirac 序列的性质，将 RGB 积分拆解为三项余弦级数，每一项只需查表一次
3. **多尺度渲染支持** — 薄膜厚度可以被空间预过滤，避免走样
4. **参数化艺术控制** — 提供物理直观的参数接口

## 三、Airy 反射公式（论文第 3 节）

### 3.1 物理构型

```
      介质 1 (空气, η1=1)
   ┌──────────────────────────── 界面 1-2
   │  介质 2 (薄膜, η2, 厚度 d)
   └──────────────────────────── 界面 2-3
      介质 3 (基底, η3 + iκ3，可以是介质或导体)
```

薄膜厚度 d 在纳米量级（通常 100~800nm），基底可以是光滑或粗糙的。

### 3.2 Airy 求和

薄膜的复反射系数是所有阶次反射光路的叠加（Figure 4）：

$$r = r_{12} + \frac{t_{12} r_{23} t_{21} e^{i\phi}}{1 - r_{21} r_{23} e^{i\phi}}$$

其中：
- $r_{ab}$ — 从介质 a 到 b 的复反射系数（含振幅和相位）
- $t_{ab}$ — 从介质 a 到 b 的实透射系数（透射无相位变化）
- $\phi = 2\pi \nu D$ — 相邻阶次间的相位差
- $D = 2\eta_2 d \cos\theta_2$ — 一阶光程差（OPD）

Airy 反射率定义为 $R = \frac{1}{2}(|r_\perp|^2 + |r_\parallel|^2)$。

### 3.3 光谱积分问题

渲染引擎最终需要的是每个光谱通道 j（如 RGB 的 R、G、B）的反射率：

$$R_j = \int R(\lambda) \cdot \frac{s_j(\lambda)}{\int s_j} d\lambda$$

直接在每个像素做这个积分（光谱采样）在厚度可变时不可行。这就是**本文要解决的核心问题**。

## 四、解析光谱预积分（论文第 4 节）— 核心创新

### 4.1 Airy 反射的傅里叶级数形式

将 Airy 反射项展开为傅里叶级数：

$$R = C_0 + 2\sum_{m=1}^{\infty} C_m \cos(m\phi)$$

其中 $C_0$ 和 $C_m$ 是解析闭式：

$$C_0 = R_{12} + \bar{R}, \quad C_m = (R_{23}R_{21})^m \cdot \frac{\bar{R} - T_{12}T_{21}}{R_{23}R_{21}}$$

物理含义：
- $C_0$ — DC 分量，对应所有同阶光路对的干涉（等价于忽略相干性的 Fresnel 反射）
- $C_m$ — 第 m 次谐波振幅，对应阶次差为 m 的光路对之间的干涉

### 4.2 频域积分

关键观察：相位 $\phi = 2\pi\nu D$ 线性依赖于频率 $\nu = 1/\lambda$。利用 Parseval 定理将光谱积分转到频域：

$$R_j = \int\hat{R}(\xi) \cdot \overline{\hat{S}_j(\xi)} d\xi$$

由于 $\hat{R}$ 是 Dirac 序列（$\hat{R}(\xi) = C_0\delta(\xi) + \sum C_m e^{im\delta_2}\delta(\xi - mD) + ...$），积分直接解析：

$$R_j = C_0 + 2\sum_{m=1}^{\infty} C_m \begin{bmatrix} \cos(m\delta_2) \\ \sin(m\delta_2) \end{bmatrix} \cdot \begin{bmatrix} \alpha_j(mD) \\ \beta_j(mD) \end{bmatrix}$$

其中 $\alpha_j, \beta_j$ 是缩放敏感函数 $\hat{S}_j$ 在频率 $mD$ 处的实部和虚部。

### 4.3 实用实现

1. **预计算**：将 $\alpha_j, \beta_j$ 存入一张 $N \times 2$ 的三通道纹理（N 为频率分辨率）
2. **运行时**：对于给定的 OPD $D$，只需 $m$ 次纹理查询（通常 $m \leq 3$ 就足够）
3. **Gaussian 近似**（进一步简化）：用 Gaussian 拟合 $\hat{S}_j$，只需几个参数即可在 GLSL 中直接求值

### 4.4 精度验证

原论文 Figure 7 在 CIE x 色度空间做了详细对比：
- **Naive 方法**（每通道一个波长采样）：颜色严重过饱和，曲线跑出 CIE RGB 色域
- **Smits & Meyer (1992)**：颜色更柔和但曲线与真值有明显偏移
- **本文方法（m=1 或 m=3）**：曲线几乎与真值重合

## 五、实际考量（论文第 5 节）

### 5.1 参数控制

| 参数 | 符号 | 物理含义 | 视觉效果 |
|------|------|---------|---------|
| 法向光程差 | $D_{inc} = 2\eta_2 d$ | 薄膜光学厚度 | 增大 → 更多颜色纹理，趋近厚层无虹彩 |
| 薄膜折射率 | $\eta_2$ | 薄膜介质折射率 | 增大 → 色彩饱和度和强度增加 |
| 基底折射率 | $\eta_3, \kappa_3$ | 基底材质属性 | 介质基底颜色柔和，导体基底更鲜艳 |

重要设计：当 $D_{inc} \to 0$，强制 $\eta_2 \to 1$ 使薄膜"消失"（退化为单界面反射）。

### 5.2 多尺度渲染

薄膜厚度可以在空间上变化。通过将反射率与高斯厚度分布做卷积，实现了厚度的预过滤：

- 预计算时对不同方差的高斯核生成 mip-map
- 运行时根据局部厚度变化率选择合适的 mip 层级
- 有效消除远处物体因薄膜高频变化产生的走样

### 5.3 实时近似

对于光滑材料（$\alpha \approx 0$），Airy 反射项只需在镜面方向求值一次，结合预过滤环境贴图即可实时渲染。对于粗糙材料，这种简化会产生过饱和颜色。

## 六、论文结果（论文第 6 节）

### 6.1 性能

| 方法 | 相对耗时 | 说明 |
|------|---------|------|
| 无虹彩 | 1.0× | 仅使用 Fresnel 反射项 |
| Naive RGB | ~1.2× | 每通道一个波长采样 |
| 光谱真值 | ~6× | 128 采样点/波段 |
| 本文方法 (m=2) | ~1.3× | 解析预积分，误差极小 |

### 6.2 效果展示

- 在粗糙皮革上添加微妙虹彩 → 提高真实感
- 金属表面氧化层 → 角度变化时显现绿色和紫色
- 汽车漆特效颜料 → 空间变化的虹彩纹理
- 透明厚板 → 透射和反射的组合效果
- 多层散射 → 前后两个透明界面各自产生干涉

## 七、与 Kim2012 的关系及对我们的意义

### 7.1 两篇论文的关系

| 维度 | Kim2012 | Belcour2017 |
|------|---------|------------|
| 发表 | 2012, 小型会议 | 2017, SIGGRAPH |
| 物理模型 | 薄膜干涉（完整公式） | 薄膜干涉 + Airy 求和 + 光谱预积分 |
| 基底 | 仅介质（空气） | 介质 + 导体（金属） |
| 光谱处理 | 直接 RGB 3 波长采样 | 解析预积分（m 次纹理查询） |
| 偏振 | 有（s/p 分量） | 有（更完整的相移处理） |
| 多次反射 | 隐含在公式中 | 明确建模所有阶次 |
| 粗糙度支持 | 无 | 可与任意微面元分布组合 |
| 多尺度 | 无 | 厚度预过滤避免走样 |
| 实现方式 | GLSL 直接计算 | 查表 + 级数截断 |

### 7.2 对当前项目的作用

**我们现在用的就是 Kim2012 的简化方案**：在 `kim2012Iridescence()` 中直接对 R(615nm)、G(535nm)、B(465nm) 三个波长计算 `thinFilmReflectance()`。这是典型的"Naive RGB"方法——计算量小但存在光谱混叠（过饱和、错误颜色）。

Belcour2017 提供了**升级路线**：

| 阶段 | 方案 | 效果 | 实现难度 |
|------|------|------|---------|
| 当前 | 3 波长直接采样 (Kim2012) | 基础虹彩，可能过饱和 | 已完成 |
| 中期 | 解析预积分 m=1 近似 | 颜色准确，无光谱混叠 | 中等 |
| 完整 | 解析预积分 m=3 + 厚度预过滤 | 高精度 + 多尺度 | 较高 |

### 7.3 实现建议

中期方案的具体步骤：
1. **离线生成 $\hat{S}_j$ 纹理**：用 CIE RGB 或 XYZ 敏感函数的傅里叶变换制成 1D 查找纹理（256×2 足够）
2. **修改 shader**：在 `main()` 中计算 OPD $D = 2\eta_2 d \cos\theta_2$，然后查表取 $\alpha_j(D), \beta_j(D)$
3. **替换 `kim2012Iridescence()`**：用 $R_j = C_0 + 2\sum_{m=1}^3 C_m[\cos(m\delta_2)\alpha_j(mD) - \sin(m\delta_2)\beta_j(mD)]$ 替代当前的 3 波长计算
4. **保留晃动模拟**：噪声扰动 d → 扰动 D，其余流程不变

### 7.4 注意事项

1. Belcour2017 的基底材料参数比 Kim2012 更复杂（$\eta_3, \kappa_3$ 随波长变化），初期可先用恒定介质的简化配置
2. 当前项目是肥皂泡（薄膜+真空基底），相当于 $\eta_3 = 1$ 的特例，公式会简化不少
3. 预积分纹理和 m=1~3 的级数求和开销远小于实时 3 波长全公式计算，反而可能更快
4. Kim2012 的薄膜干涉公式（Eq 3-5）和 Belcour2017 的 Airy 求和（Eq 3）物理上是同一件事的不同表达；Belcour2017 的优势在于正确处理了光谱预积分
